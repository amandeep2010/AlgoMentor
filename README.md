# 🧠 AlgoMentor: The Socratic DSA Learning Platform

AlgoMentor is an interactive, AI-driven web application designed to act as your personal, world-class competitive programming coach. Featuring a custom-built, sleek User Interface, this platform makes mastering Data Structures and Algorithms (DSA) highly accessible. 

Instead of functioning as a simple code generator, AlgoMentor strictly utilizes the Socratic method to guide you toward optimal solutions, prioritizing pedagogy and logical intuition over raw answer retrieval.

## ✨ Features
- **Interactive User Interface:** A seamless, distraction-free environment optimized for complex technical discussions, code block rendering, and algorithm tracing.
- **Strict Socratic Guidance:** The AI provides targeted hints, conceptual diagrams, and clarifying questions instead of direct solutions.
- **The 5-Step Instructional Framework:**
  1. Constraint & Edge-Case Analysis
  2. Brute Force Validation
  3. Systematic Optimization (e.g., Two Pointers, DP, Graphs)
  4. Dry Runs & Pseudocoding
  5. Clean Implementation & Complexity Analysis
- **Domain Restriction:** The system is strictly locked to answering DSA, complexity, and competitive programming queries to keep your learning focused and conserve tokens.
- **Language Agnostic:** Capable of validating logic across multiple ecosystems, optimized heavily for standard workflows in C++, Python, and Java.

## 💡 Example Interaction
**User:** "How do I reverse a linked list in O(N) time?"  
**AlgoMentor:** "Let's break this down. Before we write any code, imagine you have three pointers: `prev`, `current`, and `next`. If `current` is pointing to the head of the list, what is the very first connection we need to break and redirect? Take a guess at what `prev` should initially be set to."
